# U: 사용성 (Usability)

모호한 응답이나 반복 질문 등 사용성 측면의 문제 상황에서 MISO의 대응을 확인하고, PRD까지 완주합니다.

## 테스트 케이스

| ID | 테스트 | 수행 방법 | 완료 조건 |
|----|-------|----------|----------|
| U-01 | 모호한 응답 | **중간에** "그냥요", "몰라요" 등 모호한 답변 → MISO 명확화 요청 → 이후 진행 | PRD 생성 완료 |
| U-02 | 반복 질문 | **중간에** 같은 질문 계속 반복 → MISO 대응 확인 → 이후 진행 | PRD 생성 완료 |
| U-03 | 이해 확인 | **중간에** "뭐라고요?", "다시 설명해줘" → MISO 재설명 → 이후 진행 | PRD 생성 완료 |

## U-01: 모호한 응답

구체적이지 않고 모호한 답변을 했을 때의 처리를 확인합니다.

### 시나리오
```
Turn 1~2: 정상 진행
Turn 3: form 질문에 모호한 답변
  예: "그냥요", "몰라요", "아무거나요", "적당히요", "대충요"
Turn 4: MISO 명확화 요청
  - 구체적 예시 제공
  - 선택지 다시 제시
  - 유도 질문
Turn 5~N: 사용자가 구체적 답변 → 정상 진행 → PRD 완주
```

### 확인 사항
- 모호함을 인식하는가?
- 구체화를 유도하는가?
- 도움이 되는 예시를 제공하는가?

## U-02: 반복 질문

사용자가 같은 질문을 계속 반복할 때의 처리를 확인합니다.

### 시나리오
```
Turn 1~2: 정상 진행
Turn 3: "이게 뭐예요?"
Turn 4: MISO 설명
Turn 5: "근데 이게 정확히 뭐예요?" (반복)
Turn 6: MISO 다른 방식으로 재설명
Turn 7: "그러니까 이게 뭐냐고요" (또 반복)
Turn 8: MISO 구체적 예시로 설명
Turn 9~N: 사용자 이해 → 정상 진행 → PRD 완주
```

### 확인 사항
- 반복 질문을 인식하는가?
- 다양한 방식으로 설명하는가?
- 인내심을 유지하는가?

## U-03: 이해 확인

사용자가 이해하지 못했을 때 재설명 요청에 대한 대응을 확인합니다.

### 시나리오
```
Turn 1~2: 정상 진행 (MISO 설명)
Turn 3: "뭐라고요?", "무슨 말이에요?", "이해가 안 돼요"
Turn 4: MISO 다시 설명
  - 더 쉬운 용어 사용
  - 비유나 예시 추가
  - 단계별 설명
Turn 5~N: 사용자 이해 → 정상 진행 → PRD 완주
```

### 확인 사항
- 이해 부족을 인식하는가?
- 더 쉽게 재설명하는가?
- 예시를 효과적으로 사용하는가?

## 모호한 표현 예시

| 상황 | 모호한 표현 |
|-----|----------|
| 주요 사용자 질문 | "그냥 사람들이요", "아무나요", "모두요" |
| 핵심 기능 질문 | "좋은 거요", "필요한 거요", "다 넣어주세요" |
| 목표 질문 | "잘 되면 좋겠어요", "편하게요", "대충 괜찮게" |
| 문제 질문 | "불편해요", "안 좋아요", "그냥 문제 많아요" |
