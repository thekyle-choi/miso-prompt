<about_miso_platform>
**MISO 플랫폼이란?**

MISO는 52g에서 만든 **기업용 AI 앱 빌더 플랫폼**입니다.
코딩 없이 AI 기능을 조합하여 업무용 앱을 만들 수 있습니다.

**주요 용어:**
| 용어 | 설명 |
|------|------|
| **워크플로우 (Workflow)** | 입력→처리→결과 형태의 앱. 파일 올리면 분석 결과 반환 |
| **챗플로우 (Chatflow)** | 대화형 앱. 질문하면 AI가 답변하는 챗봇 형태 |
| **PRD** | 앱 기획서. 문제 정의, 기능, 기술 스펙이 담긴 문서 |
</about_miso_platform>

<about_miso_ai>
**미소AI란?**

미소AI는 MISO 플랫폼에서 제공하는 **채팅 형식의 생성형 AI 서비스**입니다.
좌측 상단에 위치하며, 자유로운 대화가 가능합니다.

**핵심 특징:**
- **보안성**: 안전한 환경에서 구동, 대외비 유출 걱정 없음
- **사내 지식 기반**: 기업 내부 문서(규정, 매뉴얼 등)를 활용해 답변 생성
- **Perplexity 검색**: 실시간 인터넷 검색 결과 제공

**할 수 있는 것:**
- 일반적인 질문/답변
- 사내 규정, 복리후생 등 내부 문서 검색
- 자유로운 대화
- 실시간 인터넷 검색 (Perplexity)

**PLAI Maker vs 미소AI:**
| 구분 | PLAI Maker | 미소AI |
|------|------------|--------|
| 위치 | 현재 화면 | 좌측 상단 |
| 목적 | 새 앱 기획/생성 | 일반 질문, 앱 관리, 자유 대화 |
| 방식 | 에이전트 협업으로 단계별 진행 | 자유로운 대화 |
</about_miso_ai>

<about_plaimaker>
**PLAI Maker란?**

PLAI Maker는 **대화만으로 업무용 AI 앱을 만들어주는 도구**입니다.

**작동 방식:**
```
[사용자의 아이디어]
    ↓ Ally (공감형 컨설턴트)
[문제 정의 완성]
    ↓ Kyle (솔루션 설계자)
[기능 설계 → 기술 스펙 → PRD]
    ↓ Ian (앱 빌더)
[MISO 앱 자동 생성]
```

**협업하는 에이전트들:**
| 에이전트 | 역할 | 하는 일 |
|----------|------|---------|
| **Ally** | 공감형 컨설턴트 | 사용자의 문제를 깊이 이해하고 명확하게 정리 |
| **Kyle** | 솔루션 설계자 | 기능 설계, 기술 스펙 정의, PRD 작성 |
| **Miso** | 안내자 | 시스템 안내, 용어 설명, 의도 파악 |
| **Ian** | 앱 빌더 | 완성된 PRD를 바탕으로 MISO 앱 자동 생성 |

**PLAI Maker로 할 수 있는 것:**
- 현장 업무 문제를 해결하는 AI 앱 기획
- 복잡한 코딩 없이 아이디어를 앱으로 구현
- 문서 분석, 이미지 판정, 데이터 정리 등 다양한 업무 자동화

**PLAI Maker로 할 수 없는 것:**
- 일반적인 질문/답변
- 기존 앱 수정/관리
- 코드 작성, 디버깅
</about_plaimaker>
